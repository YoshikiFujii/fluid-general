# FluidGeneral リリース方法

## 発行

### Windows向けの発行
dotnet publish fluid-general.Avalonia/fluid-general.Avalonia.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:PublishReadyToRun=true

### mac向けの発行（Apple Silicon）
dotnet publish fluid-general.Avalonia/fluid-general.Avalonia.csproj -c Release -r osx-arm64 --self-contained true

### mac向けの発行（Intel）
dotnet publish fluid-general.Avalonia/fluid-general.Avalonia.csproj -c Release -r osx-x64 --self-contained true



## インストーラーの作成

### Windows向け
innoSetupを利用します。インストールしてください。  
同ディレクトリのwindowsフォルダのFluidGeneral.issファイルをアプリで開き、Build>Compile
ドキュメントフォルダのInno Setup Buildsフォルダにインストーラーが作成されます。

### Mac向け
1. .appフォルダを手動で組み立てる  

FluidGeneral.app/
└── Contents/
    ├── Info.plist (メタデータファイル)
    ├── MacOS/
    │   └── (dotnet publish で出力された全てのファイル・フォルダ)
    └── Resources/
        └── Icon.icns (アプリアイコン)

.plistの内容
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>CFBundleDevelopmentRegion</key>
    <string>ja_JP</string>
    <key>CFBundleExecutable</key>
    <string>fluid-general.Avalonia</string> <!-- 実行exe（拡張子なし） -->
    <key>CFBundleIdentifier</key>
    <string>com.fluid.general</string>
    <key>CFBundleInfoDictionaryVersion</key>
    <string>6.0</string>
    <key>CFBundleName</key>
    <string>Fluid General</string>
    <key>CFBundlePackageType</key>
    <string>APPL</string>
    <key>CFBundleShortVersionString</key>
    <string>1.0</string>
    <key>CFBundleSignature</key>
    <string>????</string>
    <key>LSMinimumSystemVersion</key>
    <string>10.12</string>
    <key>NSPrincipalClass</key>
    <string>NSApplication</string>
</dict>
</plist>

2. 権限  
/FluidGeneralセットアップ関係/mac/でcmdで以下を実行  
chmod +x FluidGeneral.app/Contents/MacOS/fluid-general.Avalonia

3. dmg作成(M1/M2/M3 Macの場合)  
/FluidGeneralセットアップ関係/mac/でcmdで以下を実行
hdiutil create -volname "Fluid General" -srcfolder FluidGeneral.app -ov -format UDZO FluidGeneral.dmg
